PROJECT OVERVIEW
========================================

This is a complete Windows XP theme transformation package for Windows 11.
It provides automated scripts to transform your Windows 11 system to look
exactly like Windows XP while maintaining the Windows 11 kernel and all
modern functionality.

KEY FEATURES:
-------------
- One-click theme transformation
- Automatic backup and restore system
- 30-second test mode
- 500+ authentic Windows XP icons
- Complete XP visual theme
- Authentic XP sounds
- Classic XP wallpaper
- 100% reversible
- Safe - no system files modified

========================================
SCRIPT DETAILS
========================================

run.ps1:
------------------
- Main transformation script
- Requires Administrator privileges
- Creates automatic backup
- Applies wallpaper, colors, cursor, sounds
- Configures Windows XP visual theme
- Disables Windows 11 modern features
- Enables classic context menu
- Restarts Explorer automatically
- ~30-60 seconds execution time

back_to_win11.ps1:
---------------------------
- Restoration script
- Requires Administrator privileges
- Restores from backup
- Reverts all visual changes
- Re-enables Windows 11 features
- Restores modern context menu
- Restarts Explorer automatically
- ~20-30 seconds execution time

run-test.ps1:
----------------------
- Test mode script
- Requires Administrator privileges
- Applies theme temporarily
- 30-second countdown timer
- Automatic restoration
- Perfect for trying before committing

install_icons.ps1:
---------------------------
- Icon installation helper
- Requires Administrator privileges
- Copies icons to user directory
- Configures desktop icons
- Creates detailed installation guide
- Opens icons folder automatically
- Provides conversion instructions

START_HERE.bat:
------------------------
- Interactive menu launcher
- User-friendly interface
- Colored text interface
- One-click access to all features
- No PowerShell knowledge required
- Administrator elevation support
