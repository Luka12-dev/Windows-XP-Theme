========================================
SCRIPTS & TOOLS
========================================

1. run.ps1
----------
Purpose: Apply Windows XP theme permanently
Features:
- Automatic backup creation
- Applies wallpaper, colors, cursor, sounds
- Configures visual effects
- Disables Windows 11 rounded corners
- Enables classic context menu
- Restarts Explorer automatically
- Administrator privilege check
- Error handling and logging
- Progress indicators

2. back_to_win11.ps1
--------------------
Purpose: Restore Windows 11 default theme
Features:
- Restores from automatic backup
- Reverts all visual changes
- Restores original wallpaper
- Restores Windows 11 colors
- Re-enables transparency
- Restores modern context menu
- Restores original sounds and cursors
- Cleans up changes completely
- Administrator privilege check

3. run-test.ps1
---------------
Purpose: Test theme for 30 seconds then auto-restore
Features:
- Perfect for trying before committing
- Applies full XP theme
- 30-second countdown timer
- Automatic restoration to Windows 11
- No manual intervention needed
- Safe testing environment
- Temporary backup management

4. install_icons.ps1
--------------------
Purpose: Install and configure XP icons
Features:
- Copies 300+ icons to user directory
- Configures desktop icon paths
- Sets up icon registry entries
- Creates detailed installation guide
- Opens icons folder automatically
- Provides conversion instructions
- Includes third-party tool recommendations

5. START_HERE.bat
-----------------
Purpose: Easy-to-use menu launcher
Features:
- User-friendly text menu interface
- No PowerShell knowledge needed
